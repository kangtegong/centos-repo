How to enable CentOS repositories only from this mirror

Add following to your /etc/hosts file. 
59.29.249.221 mirrorlist.centos.org
Test that 'mirrorlist.centos.org' resolves to address from /etc/hosts. 
# ping -c2 mirrorlist.centos.org
PING mirrorlist.centos.org (59.29.249.221) 56(84) bytes of data.
64 bytes from mirrorlist.centos.org (59.29.249.221): icmp_seq=1 ttl=62 time=9.14 ms
64 bytes from mirrorlist.centos.org (59.29.249.221): icmp_seq=2 ttl=62 time=11.8 ms
